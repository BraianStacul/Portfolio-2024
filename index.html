SERVICIOS

HTML (HyperText Markup Language) es el lenguaje estándar utilizado para crear y diseñar páginas web. Permite estructurar el contenido de una página mediante etiquetas que indican cómo se debe mostrar cada elemento, como texto, imágenes, enlaces, etc.

CSS (Cascading Style Sheets) es un lenguaje de diseño utilizado para controlar la presentación y el estilo de una página web. Permite definir el aspecto visual de los elementos HTML, como el color, la fuente, el tamaño, el espaciado, etc.

JavaScript es un lenguaje de programación utilizado para crear interactividad en páginas web. Permite añadir funcionalidades dinámicas y comportamientos especiales a los elementos HTML, como animaciones, validaciones de formularios, efectos visuales, etc.

GitHub es una plataforma de desarrollo colaborativo que utiliza el sistema de control de versiones Git. Permite a los desarrolladores almacenar, gestionar y colaborar en proyectos de software de forma eficiente.
